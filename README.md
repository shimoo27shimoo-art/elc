# المجمع التعليمي للغات بالإسماعيلية

موقع شامل للمجمع التعليمي للغات بالإسماعيلية يتضمن:

## المميزات الرئيسية:
- ✅ نظام إدارة الطلاب والمعلمين
- ✅ إدارة الجداول الدراسية
- ✅ نشر الأخبار والإعلانات
- ✅ معرض الصور
- ✅ لوحة تحكم إدارية
- ✅ دعم اللغة العربية الكامل

## الملفات المرفقة:
1. **database.sql** - قاعدة البيانات والجداول والبيانات الأولية
2. **includes/config.php** - إعدادات الاتصال بقاعدة البيانات
3. **includes/db.php** - فئة الاتصال بقاعدة البيانات
4. **includes/functions.php** - الدوال الأساسية
5. **admin/dashboard.php** - لوحة التحكم الإدارية
6. **css/style.css** - التصميم والأنماط
7. **js/main.js** - البرامج النصية الجانبية للعميل

## التثبيت:
1. استيراد ملف database.sql إلى قاعدة البيانات
2. تحديث بيانات الاتصال في includes/config.php
3. تشغيل الموقع على الخادم المحلي
